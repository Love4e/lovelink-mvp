# LoveLink MVP

3D React + Supabase dating MVP (PWA-ready).

## Run locally

```bash
npm i
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Env (optional, for Supabase)

Create a `.env` file with:

```
VITE_SUPABASE_URL=https://YOUR.supabase.co
VITE_SUPABASE_ANON_KEY=YOUR-ANON-KEY
```
